# 🌟 Portfolio Website

> A portfolio website for Nitin Thakur, showcasing my work, skills, and professional journey.

## ✨ Overview

Welcome to my personal portfolio website! This project serves as a comprehensive showcase of my professional work, developer skills, and creative projects.


## 🚀 Technologies Used

**Next.js (16.1.6)** and [Aceternity UI](https://ui.aceternity.com/) component Library. I opted for this as I wanted to host freely on vercel and also aceternity is a react only framework which I wanted to use for any one of my projects for sometime.

**Frontend**
-  **Aceternity UI** as Frontend Component Library.
- Tailwind CSS (v3) for styling and layout

**Build Tools & Optimization**
- Turbopack build tool
- PostCSS for CSS processing
- Image optimization and lazy loading using next/image
- Minification and bundling using next-js turbo

**Deployment & Hosting**
- Vercel for Deployement

## 🎨 Design Philosophy

- **Clarity**: Clean typography and well-organized content hierarchy
- **Performance**: Lightweight assets and optimized code
- **Accessibility**: Inclusive design for all users
- **Professionalism**: Polished presentation that reflects attention to detail

## 🚦 Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn or any other package manager
- Git

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/epochnt/portfolio-snt.git
   cd portfolio-website
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   pnpm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   # or
   pnpm run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:3000` to view the website

### Build for Production

```bash
npm run build
# or
pnpm run build
```

The optimized files will be generated in the `next/` directory.
